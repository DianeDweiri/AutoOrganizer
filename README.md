# File Organizer Automation

This project automatically organizes files from a source folder into categorized folders and sends a daily email report.

## Features

- Organizes files by type (images, videos, PDFs, ZIPs, programs, etc.)
- Handles duplicates automatically
- Sends an email report after each run
- Logging for all actions
- Configurable through `config.json`
- Cross-platform (Windows, Mac, Linux)
- Scheduled automatically every day
